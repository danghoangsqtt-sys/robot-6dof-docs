# Tài Liệu Chuyên Sâu: Điều Khiển Cánh Tay Robot 6-DOF

> **Tác giả:** DHsystem 2026  
> **Phiên bản:** 2.0 — Smooth Movement

Tài liệu hướng dẫn chuyên sâu về lập trình, đi dây và tối ưu hóa chuyển động cho cánh tay robot 6 bậc tự do sử dụng mạch **ESP32** và chip băm xung **PCA9685**.

## 📋 Nội dung

- Phân tích phần cứng (ESP32, PCA9685, Servo)
- Cảnh báo nguồn điện & an toàn
- Sơ đồ đấu dây I2C
- Mã nguồn điều khiển C++ Arduino
- Giải thích mã nguồn chi tiết
- Gỡ lỗi (Troubleshooting)
- Tài liệu tham khảo

## 🚀 Cách sử dụng

Mở file `index.html` trong trình duyệt để xem tài liệu.

## 🛠️ Công nghệ

- HTML5 / CSS3 (Vanilla)
- Highlight.js (Syntax highlighting)
- Google Fonts (Inter, JetBrains Mono)
